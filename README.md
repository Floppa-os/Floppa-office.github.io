# Руководство по использованию Floppa Office

Добро пожаловать в **Floppa Office** — простой и удобный офисный пакет для работы с документами прямо в браузере!

Адрес вашего офиса:  
**https://floppa-os.github.io/Floppa-office.github.io/**

## 1. Начало работы

1. Откройте ссылку выше в любом современном браузере (Chrome, Firefox, Edge, Safari).
2. Вы попадёте на главную страницу с меню приложений.

## 2. Доступные приложения

В Floppa Office есть 4 инструмента:

- **Текстовый редактор** — для создания и редактирования текстов.
- **Таблицы** — для работы с табличными данными.
- **Презентации** — для создания слайдов.
- **Базы данных** — для хранения и управления данными.

## 3. Как работать с каждым приложением

### Текстовый редактор

1. Нажмите **«Текстовый редактор»** в главном меню.
2. Печатайте текст в большом поле.
3. Чтобы сохранить:
   - Нажмите кнопку **«Сохранить»**.
   - Текст автоматически сохранится в браузере.
4. При следующем открытии текст загрузится автоматически.

*Советы:*  
- Используйте `Ctrl + S` (или `Cmd + S` на Mac) для быстрого сохранения.
- Текст доступен только на этом устройстве (не синхронизируется).

### Таблицы

1. Выберите **«Таблицы»** в меню.
2. Вводите данные в ячейки таблицы.
3. Для сохранения:
   - Нажмите **«Сохранить»**.
   - Данные останутся в браузере.

*Возможности:*  
- Редактируйте любую ячейку.
- Изменения сохраняются мгновенно при вводе.

### Презентации

1. Перейдите в **«Презентации»**.
2. В левой панели — список слайдов.
3. Чтобы выбрать слайд: кликните на него в списке.
4. Редактируйте:
   - **Заголовок** — кликните и печатайте.
   - **Текст слайда** — аналогично.
   - **Заметки** — в поле ниже.
5. Для управления:
   - **«+ Добавить слайд»** — создаёт новый слайд.
   - **«− Удалить слайд»** — удаляет текущий.
   - **«Сохранить презентацию»** — сохраняет всё в браузере.

*Важно:*  
- Все слайды хранятся локально (только на вашем устройстве).

### Базы данных

1. Откройте **«Базы данных»** из меню.
2. В левой панели — список таблиц.
3. Чтобы работать с таблицей:
   - Кликните на её название для выбора.
   - Редактируйте строки прямо в ячейках.
4. Для управления:
   - **«+ Добавить строку»** — новая запись.
   - **«− Удалить строку»** — удаление текущей.
   - **«Сохранить таблицу»** — фиксация изменений.
   - **«Создать новую таблицу»** — открывает форму для ввода названия и схемы.

*Примечание:*  
- Данные хранятся только в вашем браузере.
- Для новой таблицы укажите название и структуру (пример схемы: `[{"name":"ID","type":"int"}, {"name":"Имя","type":"text"}]`).

## 4. Сохранение данных

**Важно!** Все данные сохраняются **локально** в вашем браузере (через `localStorage`). Это значит:

- Они доступны только на этом устройстве.
- При очистке кэша браузера данные удалятся.
- Нет облачной синхронизации.

**Рекомендации:**  
- Регулярно нажимайте **«Сохранить»** в каждом приложении.
- Для резервного копирования: скопируйте текст/таблицы в отдельный файл.

## 5. Возврат в главное меню

В любом приложении есть ссылка:  
**«← Вернуться в меню»** — нажмите её, чтобы вернуться на главную страницу.

## 6. Поддержка

Если что‑то не работает:

1. Обновите страницу (`F5` или кнопка обновления в браузере).
2. Очистите кэш (если проблемы сохраняются).
3. Проверьте интернет‑соединение (хотя Floppa Office работает офлайн).
